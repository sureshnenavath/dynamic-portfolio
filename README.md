# Dynamic Portfolio

This project is a dynamic portfolio website built by Nenavath Suresh.

## Features

- **Responsive Design:** Adapts seamlessly to different screen sizes.
- **Wave Animation:** A visually appealing wave animation on the hero section.
- **Interactive Elements:** Smooth transitions and hover effects.
- **Clean Codebase:** Well-structured and organized code using TypeScript and functional components.
- **SEO Optimized:** Includes meta tags for better search engine visibility.

## Technologies Used

- **React:** JavaScript library for building user interfaces.
- **TypeScript:** Statically-typed superset of JavaScript.
- **Tailwind CSS:** Utility-first CSS framework.
- **React Router:** For client-side routing.
- **@tanstack/react-query:** For data fetching and caching.
- **Radix UI:** For accessible UI components.
- **Lucide:** For icons.
- **Date-fns:** For date formatting.
- **Recharts:** For charts.
- **React Spring:** For animations.
- **Vite:** Build tool.
- **ESLint:** Linter.
- **TypeScript ESLint:** TypeScript linter.

## Getting Started

1. **Clone the repository:** (Git is not available in this environment)
2. **Navigate to the project directory:** `cd dynamic-portfolio`
3. **Install dependencies:** `npm install`
4. **Start the development server:** `npm run dev`
5. **Open your browser:** Navigate to `http://localhost:8080`  The application should now be running.
